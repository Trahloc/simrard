# Appearance

**Appearance** [genes](https://creatures.wiki/Gene) control which [sprites](https://creatures.wiki/Sprite) are used for a [creature](https://creatures.wiki/Creature)'s body parts. When you think of a norn 'she's got her mother's eyes', this is the type of gene that controls that inheritance. 
There is one gene for each body part, from head to tail tip of the creature - head, body, legs, arms are in all the games, and tail is a new option from [Creatures 2](https://creatures.wiki/Creatures_2) onwards. The body part can be specified from a selection of [breed slots](https://creatures.wiki/Breed_slot) - 0-9 in C1, and A to Z in C2 and beyond. Creatures may use the appearance of another species of creature - for example [grenorns](https://creatures.wiki/Grenorns). Appearance genes do not control the colouration of a creature, the [pigment](https://creatures.wiki/Pigment) genes do that. [Facial expression](https://creatures.wiki/Facial_expression) genes control which expressions Creatures use for which [drives](https://creatures.wiki/Drive).

Common edits for the appearance genes can be to make them [sex-specific](https://creatures.wiki/Sex-linked), as in the [Yin Yang Norns](https://creatures.wiki/Yin_Yang_Norn), which use different breed slots for males and females.

If it is specified in the [genome](https://creatures.wiki/Genome) to use a [sprite](https://creatures.wiki/Sprite) that you do not own, the game engine will use the sprites of the previous breed slot instead.
